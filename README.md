# Tavant RPOS Angular client

## Visual Studio Code extensions
If you are using Visual Studio Code, ensure that the following extensions are installed and active:

### EditorConfig for VS Code
```sh
ext install EditorConfig
```
Provides support for handling the ```.editorconfig``` file.

### Project Snippets
```sh
ext install
```
Provides access to snippets within the project, which can be committed to source control and shared across the development team.

## Code Snippets
The project contains 3 code snippets that are available to the whole development team.

- **ngt-page** - Creates a page component, following the recommended conventions of the project.
